# 📚 Taller de Programación — Repositorio Oficial

Bienvenido al repositorio oficial del **Taller de Programación** 👨‍💻👩‍💻

Este GitHub reúne todo el material utilizado durante el curso, incluyendo explicaciones teóricas, implementaciones de algoritmos, y ejercicios prácticos enfocados en el desarrollo de habilidades para la resolución de problemas.

El objetivo principal de este repositorio es servir como una **guía estructurada de aprendizaje**, donde cada tema visto en el taller está organizado de manera clara y progresiva.

---

# 🎯 Objetivo del repositorio

Este repositorio busca:

* Centralizar todo el contenido del taller en un solo lugar.
* Facilitar el estudio de algoritmos y estructuras de datos.
* Proveer implementaciones limpias y explicadas de cada técnica.
* Servir como apoyo para la práctica constante de programación competitiva.
* Complementar las sesiones con ejercicios reales de jueces en línea.

---

# 🗂️ Organización del repositorio

El contenido está dividido por temas, de manera que cada carpeta contiene:

* 📖 Una explicación teórica del tema
* 💻 Implementaciones en código (C++, Python u otros lenguajes)
* 🧠 Ejemplos ilustrativos

Ejemplo de estructura:

```text id="repo1"
📁 busqueda_binaria/
   ├── README.md
   ├── codigo.cpp


📁 criba_eratostenes/
   ├── README.md
   ├── implementacion.cpp


📁 programacion_dinamica/
   ├── README.md
   ├── ejemplos/

```

Como parte del taller, cada semana se realizarán **entrenamientos y prácticas** utilizando problemas seleccionados de distintos jueces en línea.

Estas practicas están diseñadas para reforzar los temas vistos en clases y mejorar la capacidad de resolver problemas.

# 📌 Recomendaciones de uso

Para aprovechar al máximo este repositorio:

* Estudia primero la explicación teórica de cada tema.
* Analiza los ejemplos antes de ver el código.
* Intenta resolver los problemas por tu cuenta.
* Revisa las soluciones solo después de haberlo intentado.
* Participa activamente en las practicas semanales.
